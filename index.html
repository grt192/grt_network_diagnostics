GRT Network Diagnostics.

A Java Application that prints diagnostics information into a JFrame for setting up during competition.

Current Features

  • Realtime status (CONNECTED|DOWN) of important hosts (ex. cRIO, our robot laptop, axis camera).
  • More will come in the future!
